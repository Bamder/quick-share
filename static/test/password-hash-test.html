密码哈希测试工具

测试说明

  • 本工具用于测试密码哈希功能是否正常工作
  • 密码将使用SHA-256算法进行哈希处理
  • 测试结果将显示在控制台和页面上
  • 点击"运行所有测试用例"按钮开始测试
  • 或在控制台中直接运行 runPasswordHashTests() 函数

如何查看控制台日志

  • 在Chrome浏览器中：按F12或右键选择"检查"，然后点击"控制台"选项卡
  • 在Firefox浏览器中：按F12或右键选择"检查元素"，然后点击"控制台"选项卡
  • 在Edge浏览器中：按F12或右键选择"检查"，然后点击"控制台"选项卡
  • 在Safari浏览器中：按Command+Option+I，然后点击"控制台"选项卡
